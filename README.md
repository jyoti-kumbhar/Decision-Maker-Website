# WellDecisionMaker

WellDecisionMaker is a lightweight decision-making web app that helps users break indecision quickly. It offers three modes for choosing an option: a standard random picker, a yes/no generator, and a visual spin-wheel experience. The app also remembers recent decisions, supports weighted choices, and keeps a simple local history for repeat visits.

## Features

- Standard choice picker for lists of options
- Yes/No mode for quick binary decisions
- Spin-wheel mode with animated selection
- Weighted options for more control over outcomes
- Persistent local storage for options, settings, and decision history
- Multilingual UI support
- Responsive, modern interface built with Astro and React

## Tech Stack

- Astro
- React
- Tailwind CSS
- Framer Motion
